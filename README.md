# 🔍 بوت البحث عن المفقودين

بوت تليجرام لمساعدة البحث عن الأشخاص المفقودين من خلال الصور.

## المميزات

- ✅ استقبال صور الأشخاص المفقودين
- ✅ تسجيل البيانات الأساسية
- 🔲 معالجة الصور (قريباً)
- 🔲 البحث والمطابقة (قريباً)

## التثبيت

### 1. استنساخ المستودع
```bash
git clone https://github.com/albed2026/-FindMissingBot.git
cd -FindMissingBot
```

### 2. إنشاء بيئة افتراضية
```bash
python -m venv venv
source venv/bin/activate  # على Linux/Mac
# أو
venv\Scripts\activate  # على Windows
```

### 3. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

### 4. إعداد متغيرات البيئة
انسخ ملف `.env.example` إلى `.env`:
```bash
cp .env.example .env
```

ثم أضف رمز البوت الخاص بك:
```
TELEGRAM_BOT_TOKEN=your_bot_token_here
```

### 5. تشغيل البوت
```bash
python main.py
```

## الحصول على رمز البوت

1. افتح تطبيق تليجرام
2. ابحث عن [@BotFather](https://t.me/botfather)
3. أرسل `/newbot`
4. اتبع التعليمات لإنشاء بوت جديد
5. انسخ الرمز وأضفه في ملف `.env`

## الأمان

⚠️ **مهم جداً:**
- **لا تشارك رمز البوت مع أحد**
- استخدم ملف `.env` لتخزين الرموز السرية
- تأكد من إضافة `.env` إلى `.gitignore`
- إذا تم الكشف عن الرمز، اذهب إلى BotFather واختر `/revoke`

## الاستخدام

1. ابحث عن البوت على تليجرام
2. اضغط `/start`
3. أرسل صورة الشخص المفقود
4. سيتم استقبال الصورة وتسجيل البيانات
